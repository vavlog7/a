# VALINYTSKYI AGENCY — mobile landing RU / UA

Готовый репозиторий для загрузки в GitHub Pages.

## Страницы

- `index.html` — русская версия сайта
- `ua/index.html` — украинская версия сайта
- `thanks/index.html` — страница после оплаты на русском
- `ua/thanks/index.html` — страница после оплаты на украинском
- `assets/author-photo.png` — исходная фотография автора без генерации
- `assets/config.js` — Telegram и ссылки оплаты

## Что реализовано

- Только мобильный макет: страница ограничена шириной 430 px.
- Исходная цветовая стилистика: чёрный / красный / белый.
- Фотография автора на первом экране и в блоке об авторе.
- Сохранены исходные блоки лендинга; добавлены программа на 5 дней и блок автора.
- В блоке автора указано: 4+ года работы с beauty-нишей.
- Тарифы: «Я всё сама / Я все сама» — 29 € вместо 100 €, VIP — 149 € вместо 250 €.
- Форма содержит только тариф, имя и контакт.
- Кнопка «Перейти к оплате» сначала отправляет контакт в Telegram-группу, затем переводит на выбранную оплату WayForPay.
- Кнопка консультации есть на лендинге, внутри формы и на страницах после оплаты.

## Осталось заполнить

Откройте `assets/config.js` и вставьте ссылку на бота консультаций:

```js
consultation: 'https://t.me/USERNAME_ВАШЕГО_БОТА'
```

## Публикация на GitHub Pages

Загрузите содержимое архива в корень репозитория и включите:

- `Settings → Pages`
- `Deploy from a branch`
- ветка `main`
- папка `/(root)`

## Безопасность

В тестовой версии токен Telegram-бота расположен в публичном файле `assets/config.js`. Перед рабочим запуском замените токен и вынесите отправку заявок на backend.
